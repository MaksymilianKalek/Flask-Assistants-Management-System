# Assistants Management System

This is a Flask application which allows you to:
- Create assistants
- Update their informations
- Delete them

## Installation

You need the following modules to run this app:

```bash
Flask
```
```bash
Flask SQLAlchemy
```
```bash
Pillow
```
```bash
Flask-Seeder
```
```bash
Flask-Migrate
```

You can install these modules via:

```bash
pip install
```

## Usage

To run an app:
```bash
python app.py
```
To seed the database:
```bash
flask seed run
```
